Welcome to JobVizzy, a mobile-friendly full stack web development project created as a team project by three coding bootcamp students:Attila Turkoz, Omer Kurban and Osame Ben Ismael.
Firstly the app requests user preferences, such as city, state, job titles and min salary to store in a JavaScript file-the infamous cookies!
Extracts hundreds and thousands of job listings to user with a single tap/click with 'Scraper' using BeautifulSoup, requests and Splinter modules of Python.
Transforms web data to open source database platform MongoDB and stores in lighter JSON format for smooth transfer.
Loads the job listings to user home page rendering tables in responsive, mobile-friendly web pages created with various Bootstrap components.

* * * FIRST MODULE: PROFILER* * *

  • User Input/Form

    • Collect user preferences with a form element
    • Run the next module with job title/city
    • HTML form element to collect info from user
    • JavaScript for interactive elements
    • Bootstrap components for design

    * * * SECOND MODULE: SCRAPER* * *

  • Scraper

    • Brings apprx 1,000 postings efficiently
    • Scrapes indeed, craigslist and careerbuilder
    • BeautifulSoup to parse HTML, request for source
    • Pandas for data mungling
    • Exports daily results in csv format if chosen

    * * * THIRD MODULE: LISTER* * *

  • Lister

    • User picks how to list results:
    • HTML on web, CSV for output or JSON as MongoDB
    • Results rendered HTML or exported to CSV for large data
    • Pandas for dataframe to HTML, Json or CSV
    • Stores data in JSON format to MongoDB open source database


    * * * FOURTH MODULE: BUBBLER* * *

  • Data Visualization

    • Gathers overall info for the selection parameters
    • Displays bubble chart with scraped values
    • Scrapes indeed salaries page to obtain average salary for the specific city-job title pair
    • Leaflet support for geo-map
    • Tooltip on interactive chart
